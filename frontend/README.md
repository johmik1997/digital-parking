# Frontend Notes

The main project documentation lives in [README.md](/home/john/Documents/medco-project/digital-parking2/README.md).

This `frontend` app is a Vue 3 + Vite client for the Digital Parking System demo. It includes:

- public smart parking demo page
- admin dashboard and management pages
- customer ordering flow
- cashier dashboard and reports

Local frontend commands:

```bash
npm install
npm run dev
```

Default local API target:

`v_API_URL=http://localhost:8280/api/medco-digital-parking/v1`
